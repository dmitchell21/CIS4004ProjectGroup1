# 📺 Anime Tracker

A full-stack MERN web application that allows users to track, manage, and organize their favorite anime.

## 🚀 Features

- User authentication (Login & Register)
- Two user roles:
  - Standard User
  - Administrator
- Track anime (add, update, delete, view)
- Save and manage personal anime lists
- Admin dashboard for full data management
- Search and filter anime

---

## 🧰 Tech Stack

- Frontend: React
- Backend: Node.js, Express.js
- Database: MongoDB
- Other: Mongoose, JWT Authentication
